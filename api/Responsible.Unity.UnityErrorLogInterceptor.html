Class UnityErrorLogInterceptor

Class used to replicate the Unity test runner behaviour of failing tests when either UnityEngine.Debug.LogError(System.Object) or UnityEngine.Debug.LogException(System.Exception) is called.

ExpectLog(LogType, Regex)

Expect a log message during execution of any subsequent test instructions. Similar to UnityEngine.TestTools.LogAssert.Expect(UnityEngine.LogType,System.Text.RegularExpressions.Regex), but needs to be called instead of that, in order to work when executing test instructions.

GetExternalResult<T>(CancellationToken)

Intercept errors, respecting and not toggling the globally mutable UnityEngine.TestTools.LogAssert.ignoreFailingMessages. Will complete with an error if either an error or exception is logged, and will never complete if neither of those happens.
