Class ScenarioAttribute

Annotates a method as a BDD-style scenario returning steps, which will be run as a test case, if the containing class is annotated with FeatureAttribute. The return type of the method must be convertible to IEnumerable<IBddStep>.

Remarks

Examples
[Scenario("Example scenario")]
