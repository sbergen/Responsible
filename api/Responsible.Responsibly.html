Class Responsibly

Main class for constructing primitive wait conditions and instructions.

Inheritance
Object
Responsibly
Namespace: Responsible
Assembly: Responsible.dll
Syntax
public static class Responsibly
Remarks

Instead of using a class like WaitFor, we try to avoid conflicting class names by having this class. It also allows you to use either Responsibly.WaitForCondition or using static Responsible.Responsibly and simply WaitFor.
